Selection Behavior (no auto-paste on arrow key)
Keyboard flow
← → moves highlight to prev/next card
Card shows "Enter to paste" hint in footer
Enter pastes the highlighted card & dismisses
Ctrl+Enter copies without pasting or dismissing
Mouse flow
Hovering a card reveals ↵ Paste + ⎘ Copy in footer
Clicking anywhere on card = paste & dismiss
Clicking ⎘ Copy = copy only, panel stays open
Right-click = context menu (Pin / Copy / Delete)
Show — Alt+V pressed
1. Window positioned at screen bottom edge
2. TranslateTransform.Y = +panelHeight
3. Storyboard: Y → 0 over 240ms
4. Easing: CubicEase Out
5. Opacity: 0 → 1 in first 80ms
6. Window stays resident (Show/Hide, not Open/Close)
Hide — Esc / focus lost / Enter paste
1. Storyboard: Y → +panelHeight over 180ms
2. Easing: CubicEase In
3. Opacity: 1 → 0 in last 60ms
4. Window.Hide() called on Completed
5. Paste action fires before animation if triggered by Enter
